# Node.js Calculator

A simple command-line calculator built using Node.js.  
It performs basic arithmetic operations like addition, subtraction, multiplication, and division.

---

## 🚀 Features

- Addition
- Subtraction
- Multiplication
- Division
- Squareroot
- Power
- quadratic equation
- User input through terminal

---

## 🛠️ Technologies Used

- Node.js
- JavaScript
- readline module
